## Running locally:
- Clone repository into your local machine 
- If opening the project in Visual Studio, hit the Green "Start" button at the top to run.
- If using PowerShell terminal, run `dotnet watch run`
- When finished, either click the Red "Stop" button at the top of Visual Studio OR type `Ctrl + C` in PowerShell terminal
